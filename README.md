# Souqvia

Souqvia is an AI-powered product finder for Algerian e-commerce platforms like Ouedkniss, Jumia, and Facebook Marketplace. It helps users discover profitable products to sell online with smart recommendations.

## Features

- Search products quickly from multiple Algerian e-commerce sites
- Smart AI recommendations for profitable products
- Multi-language support: Arabic, French, English
- Filter by price, condition, and location

## How to Use

1. Open the website: [https://yourprojectlink.vercel.app](https://yourprojectlink.vercel.app)
2. Enter a product name in the search bar
3. View recommended profitable products
4. Apply filters for better results

## Tech Stack

- React (Frontend)
- Tailwind CSS (UI)
- Gemini AI / OpenAI API (Product analysis)
